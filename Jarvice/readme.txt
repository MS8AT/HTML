# 🧠 JARVICE / J-TEC AI NODE

### Автономный AI-сервер нового поколения

---

## 📌 Описание

Jarvice / J-Tec — это автономный вычислительный узел, объединяющий:

* локальные AI-модели
* систему автоматизации
* постоянную память
* API-интерфейсы

Система реализует концепцию:

**Self-learning AI node** — самообучающийся информационный узел.

---

## ⚙️ Аппаратная конфигурация

Сервер представляет собой высокопроизводительную вычислительную платформу:

* **CPU:** 2 × Intel Xeon E5-2699 v4
* **RAM:** 256 GB DDR4
* **Storage:** 3 × NVMe 2TB (RAID0)
* **GPU:** 8 × RTX 3060
* **ОС:** Ubuntu Server

Характеристики:

* высокая параллельность вычислений
* NVMe скорость до ~4.5 GB/s
* поддержка работы с AI-моделями

---

## 🧩 Программная архитектура

Система построена как интегрированный AI-узел.

### Основные компоненты:

* **Linux** — базовая операционная система
* **Docker / сервисы** — изоляция процессов
* **n8n** — оркестрация и автоматизация
* **MySQL (J-Memory)** — хранение данных
* **vLLM / DeepSeek** — локальные языковые модели
* **Web API / Proxy** — интерфейс доступа
* **Telegram bot** — управление
* **Watchdog** — контроль и перезапуск

## 🔌 Архитектура питания

Сервер использует **двойную систему питания (Redundant Power Supply)**:

* PSU1 — основная линия
* PSU2 — резервная линия

Обеспечивает:

* непрерывную работу
* отказоустойчивость
* возможность замены без выключения
